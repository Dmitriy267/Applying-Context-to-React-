Контекст React инициализируется с помощью createContextAPI.Контекст всегда должен быть инициализирован в отдельном файле (например, src/currency-context.js или src/contexts/currency.js ), потому что мы будем повторно использовать его во всем приложении.Функция React createContextпринимает начальное значение , которое будет значением по умолчанию.
Объект Context, который мы создали ранее, предоставляет компонент Provider, который чаще всего используется где-то на верхнем уровне (например, компонент приложения) вашего приложения React, чтобы предоставить свой контекст всем дочерним компонентам (которые заинтересованы в нем) ниже .

Это означает, что мы не передаем значение через реквизит . Вместо этого мы передаем значение через контекст .

Кроме того, объект Context предоставляет компонент Consumer, который можно использовать во всех дочерних компонентах (где-то ниже компонента Provider), которым требуется доступ к контексту.Теперь наступает важный момент, когда мы переходим на использование React's useContext Hook.
Хук useContext в React принимает контекст в качестве параметра для извлечения valueиз него. Использование React Hook вместо компонента Consumer делает код более читаемым, менее подробным и не вводит промежуточный компонент (в данном случае компонент Consumer)
